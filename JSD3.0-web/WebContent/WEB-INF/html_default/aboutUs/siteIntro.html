关于2.0托管版

杭州纵通电子商务有限公司是一家隶属于浙江通策控股集团信息事业部的合资公司，合资方为浙江安信资产管理集团，注册资金500万。公司核心团队成员来自第三方支付机构、大型金融资产管理集团、互联网金融服务企业，具备强大的技术实力及丰富的行业经验。公司专注于互联网金融业务创新，侧重汽车金融、健康医疗等高成长性行业，为相关行业小微企业提供互联网金融服务。

杭州纵通电子商务有限公司旗下2.0托管版（www.zotohui.com）是专业的P2P互联网融资服务平台。平台立足自身优势，侧重汽车金融服务，医疗健康等高成长性行业融资信息服务。平台具备专业完善的贷前、贷中、贷后风控体系和技术平台，以及安全的网上交易环境和全面资金保障措施。2.0托管版以透明、严谨、高效、创新为发展宗旨，为投资人和借款人提供一个安全可信、公开透明、平等自由的P2P互联网融资服务平台。

集团背景

通策集团

通策集团（www.topchoice.com.cn）是一家拥有地产、口腔医疗、互联网、文化等多元化事业的综合性企业集团。凭借良好的价值观和信仰的力量，以满足客户需求为使命，荟集了大量专业人员，专注于各板块业务。使集团业务得获得健康发展，并在同行业内享有较高口碑。通策集团起步于1994年，目前旗下经营的几大事业部分别是：口腔事业部、不动产事业部、信息事业部、金融事业部、妇幼事业部。

通策医疗投资股份有限公司（www.tcmedical.com.cn ）于1996年在上海证券交易所挂牌上市交易，证券代码：600763。是目前国内唯一一家以医疗管理、投资管理为主营业务的主板上市公司。2010年荣登福布斯中国最具潜力中小企业排行榜；2010年7月荣获2010年中国医药最具竞争力上市公司50强，2011年5月荣登中联上市公司价值百强榜单60强。公司目前拥有杭州口腔医院、宁波口腔医院、衢州口腔医院、北京通策京朝口腔医院、河北沧州口腔医院、湖北黄石现代口腔医院、昆明口腔医院等医院和医疗服务连锁机构。

通策地产（中国）有限公司创立于1995年，在杭州城区先后成功开发了天巢花苑、正和居、墨香苑、通策广场、和睦院、钱江时代等多个精品项目，取得了巨大的商业成功。

通策集团信息事业部在集团公司专业技术的基础上，整合了第三方支付、电子商务、医疗信息化、云计算、企业信息化、移动互联网等业务。

安信集团

浙江安信资产管理集团有限公司是一家专业从事资产管理、金融担保、商品融资、汽车按揭、汽车租赁等业务的综合性金融服务企业。公司成立于2004年，注册资本为1亿元人民币，现有员工近1000人。公司总部设运营企划中心、行政人力中心、财务审计中心三大职能部门、以及汽车金融事业部、资产保全事业部等两大事业部以及控股华凯保险销售公司、安信汽车网络服务公司。 

安信源自台州，立足浙江，面向全国，致力于成为国内知名的汽车金融服务企业。目前已设立杭州、宁波、台州、温州、绍兴、金华、衢州、丽水、嘉兴、湖州、舟山等省内近30家分公司或营业网点，并在省外建立6家子公司，业务遍布浙江、山东、江西、湖南、安徽、上海和江苏全省各地，初步形成了功能齐、服务全的金融服务网络。经过十年的发展，无论是资产规模还是经营规模，安信都已经成为浙江省最大的汽车消费贷款担保机构之一，在业界和社会上享有较高的知名度和美誉度。
