晋商贷号列车

2015年4月3日，晋商贷敢为人先，与畅达传媒集团强强联手，斥巨资打造专属高铁品牌专列，并成为了P2P行业内首家冠名高铁动车组专列的品牌。它将晋商贷独特的“晋商”理财理念传递给全国各地乘客，将优质的产品奉献给中国最具购买力和消费力的高铁人群，帮他们实现财富增长。

企业简介

晋商贷是山西新晋商电子商务股份有限公司旗下全资品牌，它结合中国的社会信用状况，利用可靠的信用审核模型和先进的技术，创建了适合中国的P2P小额借贷网络平台。

务实创新、科学发展、和谐共赢，为社会创造价值，是晋商贷长期追求的目标。它为创造就业、盘活民间闲散资金，为微小企业及有资金需求的个人输送最快捷有力的资金血液。为中国数以亿计的微小企业及个人创业，实现无可限量的长期经济发展、社会和谐，作出了重大突出贡献。

晋商贷，责任金融的典范！作为山西最大的互联网金融信息服务平台，也是中国最早的一批基于网络的P2P信息服务平台，晋商贷以其诚信、透明、公平和高效的特征赢得了良好的用户口碑。
