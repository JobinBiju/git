第三方资金托管指晋商贷平台投资人的投资理财资金不经过晋商贷的银行账户，所有的资金流转均通过第三方资金托管公司进行，避免平台挪用投资人资金而给交易双方带来风险。

央行的目的也很明确，希望通过第三方资金托管的方式避免资金流入平台的银行账户形成资金池，导致资金池内资金被挪用的风险

汇潮支付在深入了解晋商贷对第三方资金托管、支付安全、结算服务等需求后，为晋商贷商户及用户量身定制的行业解决方案，为晋商贷参与者规避政策风险、确保资金安全，推动晋商贷平台有序健康的发展。

晋商贷平台通过接入该系统将投资人、借款人、平台三者的资金完全隔离，从技术层面解决了平台挪用平台交易资金的问题。
