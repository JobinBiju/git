关于汇潮支付

汇潮支付有限公司,Ecpss Co. Limited(E-commerce Payment Service Solution) 简称：汇潮支付，专注于网上支付服务的第三方支付平台， 全力为国内外接受网银，信用卡支付的商家提供世界一流的收单服务。

汇潮支付由前银行卡支付专家、前国际卡组织高级管理人员、第三方支付资深管理人员、资深国际卡风险管理专家和支付行业安全认证专家共同投资组建，致力于创建全球第一的第三方支付平台。

汇潮支付秉承"合作共赢，服务一流"的理念，全力打造成功率最高、风险最低的第三方支付平台。

汇潮支付专业提供网上支付服务，目前提供的支付服务有网银支付，国际卡支付，带银联标识的银行卡，信用卡支付；同时，为减少chargeback，公司还为商户提供国际卡风险控制系统。

汇潮支付与中国银行、中银国际卡公司、中国工商银行、中国建设银行、中国农业银行、交通银行、招商银行、中信银行、上海浦东发展银行、中国民生银行、兴业银行、中国光大银行、华夏银行、广东发展银行、深圳发展银行、上海银行、北京银行、南非银行、英格兰银行等等金融机构，以及VISA、MASTER CARD等国际组织建立紧密合作关系。

2012年6月汇潮支付取得中国人民银行颁发的《支付业务许可证》(证书编号：Z2010331000018)，公司立足于银行卡受理市场建设，积极开拓第三方支付业务，支持国内所有的银行卡，信用卡支付，公司主要业务品种包括：网上支付、网银收款、一麻袋电子钱包、企业下发、信用卡支付、酒店机票订购，网贷P2P收款等互联网上支付业务。

汇潮支付的意愿: 我们向个人和公司提供最新、最安全、最快捷、最低廉的电子支付服务，实现商业效益的最大化。

专业支付机构 用心提供服务

2012年6月，E汇通荣获了由中国人民银行颁发的《支付业务许可证》 ，获得互联网支付业务许可。支付牌照的发放有助于加强监管，有利于行业的发展，能够促进整个支付市场的繁荣。E汇通将全力进军第三方支付行业，致力于行业细分市场，为个人及企业提供简单、方便、有利、安全的综合电子支付产品与服务。

多重安全认证 保障用户利益

为了维护商户及消费者权益，E汇通由风险控制专家对商户开通和交易过程层层把关，对账户信息、密码、注册资料等进行加密保障，交易过程中信息不外泄，创造一个安全、平等、公正的支付平台。

维护金融稳定 确保资金安全

E汇通完全认同并遵守国家相关的金融安全法规和制度。在资金监管方面,E汇通依监管机关要求,将客户资金独立存放并由存管银行监督管理；制定符合金融要求的客户资金管理制度，确保操作无误；同时，E汇通根据监管要求，采用技术手段对各类洗钱和恐怖融资活动进行严格放控。
