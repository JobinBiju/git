点击注册链接
打开首页，找到广告图上方登录框中的“免费注册”链接，即可跳转至注册页面。
填写注册信息
根据要求填写好注册信息，并同意《2.0托管版用户服务协议》，最后点击“立即注册”按钮。
完成邮箱激活
信息填写完毕，系统会自动发送激活邮件至您的注册邮箱，请立即登录邮箱，完成激活验证。
进入登录页面
完成邮箱激活后，即完成了账户注册。进入登录页面，输入账户密码即可进入账户中心。
填写用户名
打开首页，找到广告图上方登录框，输入用户名与密码，点击立即登录。支持用户名、手机及邮箱登录。
登录账户成功
填写好账户密码后即可登录个人账户，在此可以看到个人信息与投资信息。
进行个人认证
点开个人账户中“我的账户”下的子菜单“支付账户”，输入个人信息进行认证。
确定开户信息
跳转至环迅支付页面，确认您的身份信息，并确认手机号码，最终确认开户。
设置账户密码
设置第三方账户的登录密码，交易密码和密码保护问题，然后点击确认。
设置银行账户
填写银行信息，包括名称与开户行所在地，支行名称。以及账号与户名。
账户开户成功
提示开户成功。
点击充值按钮
打开账户中心首页，点击充值按钮，跳转至充值界面。
填写充值金额
在出现的账户充值界面，选择充值银行，填写充值金额与验证码，点击确认提交按钮。
输入交易密码
进入环迅交易页面后，确认交易信息，并输入交易密码，点击确认后，即可完成操作。
点击我要投资
点击一级菜单中的“我要投资”链接。
选择投标项目
在出现的“精选投资列表”中，选择你要投资的项目，然后点击右侧的“我要投标”按钮。
输入投资金额
在出现的借款详情页中，在右侧“投资金额 ”栏目里，输入您的投资金额，点击确认按钮后，即完成投资。
进入我的账号
输入您的账户信息，进入我的账户页面。
查看账户收益
登录个人账户页面后，在账户首页，左侧即可看到“今日收益 ”和“累计净收益”。
点击提现按钮
登录个人账户之后，点击账户主页右侧的“提现”按钮。
填写提现金额
在账户提现页面中，输入提现金额后，点击确认。
