我的奖励

  • 静态红包

  • 动态红包

温馨提示：

1.用户注册完成三项认证之后送静态红包，平日送5元，活动日送50元。活动时间为：2015年4月16日 至2015年7月31日。推荐好友完成三项认证之后给推荐人赠送静态红包，平日0.5元，活动日1元。活动时间为：2015年5月25日至2015年6月30日。

2.被推荐人首次投资超过1万元(包含1万)，赠送推荐人千分之一的静态红包。

3.静态红包可用来提现(待收本金大于10000)，但不能用来投资。

4.投标送动态红包。月标：赠送动态红包金额 = (投资金额/1000*月份),天标：赠送动态红包金额=(投资金额/1000*（天数/30）)。

5.动态红包可用来投资，不可直接提现。月标：动态红包可使用金额=(投资金额/1000*月份),天标：动态红包可使用金额=(投资金额/1000*（天数/30）)。
