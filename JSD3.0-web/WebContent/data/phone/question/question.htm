1、为什么要进行实名认证？

晋商贷为保障用户账户资金和信息的安全，需所有投资者和借款人通过身份证绑定、手机绑定、邮箱地址绑定、银行卡绑定以及提现支付密码设置。每个实名认证环节均简单快捷，晋商贷在此申明会严格保密用户信息。

2、什么样的债权可以转让？

1. 原投资债权月标需持有3个月后并且在当期还款日5天前转出

2. 购买债权再转出，需要持有N天后并且在当期还款日5天前转出。

3、账户提现需要提现手续费吗？

为了答谢新老会员长期以来对晋商贷的支持和关注，从新版托管版本上线之日起，凡是晋商贷所有会员在新版上的正常回款，提现手续费全免。

4、我的账户被锁了怎么办？

连续输入错误密码5次，系统将会将你的账户进行锁住；账户被锁24小时内系统会自动解锁。你也可以选择跟您的专属客服进行联系，或者拨打我们的客服热线400-080-3000进行核实解锁。

5、身份证认证成功后，身份证号码可以更换吗？

不可以更换，因为第三方托管账户是不可以更改实名认证的。

6、获取短信验证码失败怎么办？

在规定时间内未获取到短信验证码，您可以及时联系您的专属客服：4000803000或者拨打我们的客服在线电话：400-080-3000。

7、提现银行卡可以绑定信用卡作为提现卡吗？

不可以

8、为什么要填写开户行支行？

银行规定跨行汇款必须要填开户行名称，名称写错或没写，提现可能会被退回，给您造成不便，为了保证资金能够安全到达您的账户，请准确填写具体提现银行卡开户行信息。

9、绑定提现银行卡需要注意什么？

1、提现银行卡不支持绑定信用卡作为提现卡，没有特殊需求，建议只绑定一个最常用的唯一提现银行卡。

2、请用户尽量绑定四大行或较大的商业银行（如：农行、工行、建行、中国银行、民生银行、招商银行、光大银行、兴业银行等），避免填写那些比较少见的银行（如农村信用社、平安银行、恒丰银行等）。否则提现资金很容易会被退票。
